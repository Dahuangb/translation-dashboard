# 综合翻译模型评估看板 (Dashboard)

这是一个基于 [Streamlit](https://streamlit.io/) 构建的翻译模型质量评估与 Bad-case 检索专业看板。本项目采用“总-分”叙事架构，支持全局表现对比、单语种下钻、错误类型诊断以及细粒度的案例检索。

## 运行环境要求
请确保您的电脑上已安装 Python 3.8 或更高版本。

## 快速启动步骤

1. **解压文件**：将本压缩包解压到您的本地文件夹中。
2. **打开终端**：在终端（Terminal / 命令行提示符）中进入解压后的文件夹目录。
3. **安装依赖**：
   在终端中运行以下命令，安装看板所需的相关 Python 库：
   ```bash
   pip install -r requirements.txt
   ```
4. **启动看板**：
   依赖安装完成后，在终端中运行：
   ```bash
   streamlit run app.py
   ```
5. **访问页面**：启动成功后，您的默认浏览器会自动弹出一个新标签页并访问 `http://localhost:8501`。如果没有自动弹出，请手动复制上述链接在浏览器中打开。

## 核心文件结构

- `app.py`：看板的主程序与前端渲染代码。
- `data/`：核心数据文件夹。包含所有图表所需的宏观统计指标数据和海量的 bad-case 结构化案例库（已转为 JSON 格式，开箱即用）。
- `parse_bad_cases.py`：数据清洗脚本（用于读取原始 CSV 并生成 `data/` 下的 JSON 文件。此脚本主要供数据更新时使用，日常查看无需运行）。
- `dashboard_summary.md`：本次看板开发的架构梳理与核心重构总结报告。
- `requirements.txt`：Python 依赖清单。

## 常见问题
- **如果遇到“命令未找到: streamlit”的报错**：
  这通常是因为 Python 环境变量未配置。您可以尝试使用绝对路径运行，或者使用 `python3 -m streamlit run app.py` 来启动服务。
- **页面数据未更新**：
  如果您替换了新的底层数据，可以点击看板右上角的菜单（⋮），选择“Clear Cache”（清除缓存），然后刷新页面即可。
