# 🎓 小学数学复习乐园 - 三年级下册（北师大版）

一个专为小学三年级下册学生设计的数学复习应用，采用儿童友好的配色和UI设计，全面覆盖北师大版教材知识点。

## ✨ 功能特点

- 📚 **全面覆盖**：北师大版三年级下册数学8个单元全部知识点
- 🎨 **儿童友好**：明亮活泼的配色、圆角设计、大字体
- 🎮 **互动练习**：每个知识点都配有选择题练习
- ⭐ **成就系统**：收集星星、解锁成就、查看学习进度
- 📱 **响应式设计**：完美适配手机、平板和电脑
- 💾 **本地存储**：学习进度自动保存

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 本地运行

```bash
npm run dev
```

打开浏览器访问 `http://localhost:3000`

### 3. 构建生产版本

```bash
npm run build
```

## 🌐 部署到 Vercel（获取公开链接）

### 方法一：使用 Vercel 网站部署（推荐）

1. 将代码推送到 GitHub 仓库
2. 访问 [vercel.com](https://vercel.com)
3. 使用 GitHub 账号登录
4. 点击 "New Project"
5. 选择你的项目仓库
6. 点击 "Deploy"
7. 等待部署完成，你将获得一个公开链接（如：`https://math-review.vercel.app`）

### 方法二：使用 Vercel CLI 部署

```bash
# 安装 Vercel CLI
npm i -g vercel

# 登录 Vercel
vercel login

# 部署项目
vercel
```

## 📖 知识点覆盖

### 第一单元：除法
- 两位数除以一位数
- 三位数除以一位数
- 商中间或末尾有0的除法
- 除法的验算

### 第二单元：图形的运动
- 轴对称图形
- 平移现象
- 旋转现象

### 第三单元：乘法
- 两位数乘整十数
- 两位数乘两位数（不进位）
- 两位数乘两位数（进位）

### 第四单元：千克、克、吨
- 认识质量单位
- 质量单位换算

### 第五单元：面积
- 面积和面积单位
- 长方形和正方形的面积计算
- 面积单位的换算

### 第六单元：分数的初步认识
- 认识分数
- 分数的大小比较
- 同分母分数的加减法

### 第七单元：数据的整理和表示
- 数据的收集与整理
- 数据的分析

### 补充：年、月、日
- 年、月、日的认识
- 24时计时法

## 🛠️ 技术栈

- **前端框架**: React 18

- **构建工具**: Vite
- **路由**: React Router v6
- **状态管理**: React Hooks + Context API
- **样式**: CSS Modules
- **部署**: Vercel
